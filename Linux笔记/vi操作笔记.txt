进入vi:
vi <文件名>
vi可以自动帮你载入所要编辑的文件或是开启一个新文件。进入vi后屏幕
左方出现波浪符号，凡是列首有该符号就代表此列目前空的。



ESC键是切换到指令模式下

vi有命令模式、插入模式（编辑模式）、末行模式 三种模式

1、命令模式
i：进入插入模式，光标前插入
I：进入插入模式，行首插入
a：进入插入模式，光标后插入
A：进入插入模式，行末插入
o:向下开辟一行，进入插入模式
O：向上.....

移动光标：
h：左移
j：下移
k：上移
l：右移
M：光标移动到中间
L：光标移动到屏幕最后一行行首
G：移动到指定行，行号 -G
w：向后一次移动一个字
b：向前一次移动一个字
{：按段移动，上移
Ctrl-d：向下翻半屏
Ctrl-u：向上翻半屏
Ctrl-f：向下翻一屏
Ctrl-b：向上翻一屏
gg：光标移动到文件开头
G：光标移动到文件末尾

删除指令
x：删除光标后一个字符，相当于 Del
X：删除光标前一个字符，相当于 Bzckspace
dd：删除光标所在行，n dd 删除指定的行数  
D：删除光标后本行所有行都删除
d0：删除光标前本行所有内容，不包括光标所在字符
dw：删除光标开始位置的字，包括光标所在字符

插销指令
u：一步一步撤销
U：一次性撤销当前所作的所有操作
Ctrl：反撤销

重复命令
.：重复上一次操作的命令

文本行移动：
>>：文本行右移
<<：文本行左移

复制粘贴
yy：复制当前行，n  yy  复制 n 行
p：在光标所在位置向下新开辟一行，粘贴

可视模式
v：按字符移动，选中文本
V：按行移动，选中文本可视模式可以配合d , y ,>> ,<< 实现对文本的编辑

替换操作
r：替换当前字符
R：替换当前行光标后的字符

分屏操作
sp：上下分屏年后可跟文件名
vsp：左右分屏，后可跟文件名（vsp  文件名）
Ctrl+w+w：在多个窗口切换
vim -On(n是分几个屏)  文件1  文件2 （左右分屏）
vim -on(n是分几个屏)  文件1  文件2 （水平分屏）
ctrl+w c ：关闭当前分屏
ctrl+w q ：关闭当前窗口，如果只剩最后一个，则退出vim

屏幕尺寸
1、增加高度
ctrl+w +
2、减少高度
ctrl+w  -
3、让所有屏的高度一致
ctrl+w =
4、左加宽度
ctrl+w >
5、右加宽度
ctrl+w <
6、右增加n宽度（如：n=30）
ctrl+w n<


查找命令
/：查找
n去找下一个
K查看函数

3、末行模式
离开vi：
:q 退出vi不保存
:q！ 强制退出vi不保存
:wq 指令则是存档在离开
:wq！强制指令则是存档在离开
set nonu ：没有数据（使vim文件没有行号）
set nu ：使vim文件有行号
